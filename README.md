# Simple Interest Calculator

## Overview
This is a simple Python program to calculate **Simple Interest** based on user input. It demonstrates basic arithmetic operations, input handling, and output formatting.

## Formula
The formula for Simple Interest is:  
SI = (P × R × T) / 100
Where:  
- **P** = Principal amount  
- **R** = Rate of interest (per annum)  
- **T** = Time period (in years)  
- **SI** = Simple Interest  

## Features
- Accepts user input for principal, rate, and time.
- Calculates simple interest accurately.
- Displays the result in a readable format.
